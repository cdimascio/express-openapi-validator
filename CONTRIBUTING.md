# contributing to express-openapi-vaidator

## Prerequisites / Setup

- Fork the repo
- Clone your copy `git clone <forked-repo>`
- `npm install`

## Run the tests

```shell
npm test
```

## Project structure

`src` contains the source code
`test` contains the tests
